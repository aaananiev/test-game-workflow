How to use
----------------------------------------------------------

1. Open a terminal in this folder and run 'npm install'

2. To collect the raw data run 'npm collectData'. 
This will generate the rawData.jsonl file.

If you want to change the number of runs per tier you will need to edit collectData.js

3. To process raw data and generate results run 'npm processData'
This will process rawData.jsonl and generate the results.json file.
